# Netflix Roulette

Проект для EPAM Training Center с использованием React, React Router, Redux, TypeScript, Webpack

(upd. API с фильмами больше не работает)

## Состав

### Установка

Первый запуск 

```
npm install
```

### Запуск проекта

Запуск приложения в dev - моде

```
npm run start
```

Запуск приложения в prod - моде

```
npm run build
```
